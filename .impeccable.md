# .impeccable.md — Design Context

## Design Context

### Users
**주 사용자: 입학 지원 학생의 학부모**
- 자녀의 서울대학교 입학 당일, 스마트폰으로 접속
- 긴장과 기대가 교차하는 고부담(high-stakes) 상황에서 사용
- 해야 할 일: 대기열 번호 받기 → 순번 확인 → 등록 완료
- 디지털에 익숙하지 않은 중·장년층도 포함될 수 있음
- 빠르고 명확한 피드백이 불안을 낮추는 핵심 역할

### Brand Personality
- **3단어**: 신뢰, 안정, 격식
- 서울대학교 공식 채널로서의 권위와 공신력
- 차갑지 않고 학부모를 배려하는 따뜻한 공식성
- 텍스트 어조: 존댓말, 명확한 안내, 불필요한 유행어 없음
- 감성 목표: "이 시스템은 믿을 수 있다"는 확신 → 불안 해소

### Aesthetic Direction
- **시각적 기준**: 서울대학교 아이덴티티 — SNU Blue(`#003B71`) 중심, 절제된 색상 팔레트
- **레퍼런스 방향**: 국내 주요 대학·공공기관의 공식 디지털 서비스 수준, 단 현대적으로 정제된 버전
- **Anti-references (절대 피할 것)**:
  - 과도한 그라디언트·네온·파티 느낌의 화려한 디자인
  - 스타트업 감성 (Y Combinator식 sans-serif 미니멀, 과감한 여백)
  - 극단적 미니멀 (정보가 너무 없어 신뢰가 떨어지는 빈 화면)
  - 정부 민원 사이트식 올드하고 관료적인 레이아웃
- **테마**: 라이트 모드 우선 (다크 모드는 선택적 지원)
- **모바일 최우선**: 학부모가 현장에서 스마트폰으로 사용 — 터치 타겟, 가독성, 한 손 조작 고려

### Design Principles

1. **신뢰를 시각화한다**
   SNU Blue 기반의 일관된 색상 체계와 여백의 균형으로 공신력 있는 기관의 느낌을 유지한다. 색상을 과하게 사용하거나 임의로 바꾸지 않는다.

2. **상태를 즉시, 명확하게 전달한다**
   대기 중 / 입장 가능 / 완료 등 현재 상태를 한눈에 알 수 있도록 설계한다. 숫자와 색상 모두 상태를 동시에 표현해야 한다.

3. **불안을 낮추는 타이포그래피**
   한글 가독성을 우선으로, 충분한 크기(최소 16px body), 명확한 위계, 과도한 약어·영문 혼용 없이 쉬운 말로 쓴다.

4. **모바일 터치에 최적화한다**
   버튼 최소 높이 56px, 탭 간격 충분히 확보, 스크롤 최소화 — 한 화면에서 핵심 행동이 완결되도록 설계한다.

5. **격식을 갖추되 친절하다**
   딱딱한 관료적 언어 대신, 존댓말로 쓰인 명확한 안내문. 오류 메시지도 비난하지 않고 다음 행동을 알려주는 방식으로 작성한다.

## Token Reference

| Token | Value | Usage |
|---|---|---|
| `snu-blue` | `#003B71` | 주요 버튼, 헤더, 강조 |
| `snu-dark` | `#00274c` | 호버 상태, 진한 강조 |
| `snu-gray` | `#E8E9EA` | 배경, 구분선 |
| `snu-text` | `#333333` | 본문 텍스트 |
| emerald | `#059669` | 성공, 입장 가능 |
| rose | `#be123c` | 경고, 마감 임박 |
| amber | `#b45309` | 주의, 안내 |

## Typography

- **Font**: `-apple-system, BlinkMacSystemFont, "Noto Sans", Helvetica, Arial, sans-serif`
- **Body min**: `text-base` (16px), `leading-relaxed`
- **Headings**: `font-bold` ~ `font-black`, `text-2xl` ~ `text-4xl`
- **Labels**: `text-xs`, `uppercase`, `tracking-[0.15em]` ~ `tracking-[0.3em]`
- **Korean copy**: 존댓말, 명확한 동사 종결, 불필요한 영문 약어 최소화

## Component Conventions

- **Card radius**: `rounded-2xl` ~ `rounded-3xl`
- **Button height**: `min-h-[56px]`, `rounded-2xl`, `w-full` on mobile
- **Shadows**: `shadow-sm` ~ `shadow-lg` (과하지 않게)
- **Icon library**: Lucide React
- **Spacing**: Tailwind 기본 스케일, 주요 padding `p-5` ~ `p-8`
